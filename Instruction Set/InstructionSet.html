How to Build a Webpage

An instruction set for creating your first web page with HTML and CSS.

Introduction

This instruction set guides novice users through learning how HTML and CSS can be used to create a webpage. HTML(hypertext markup language) and CSS (___ Style Sheet) are the foundation of any webpage that you see across the internet.

  • Total Time: __ minutes
  • Skills Required: Basic computer literacy and ability to manage files in the file exploror.
  • Required Materials:You will need a professional code editor like VS Code, a modern web browser like chrome, and a folder on your local machine to store your HTML and CSS files.

Step 1: Environment Setup

1. Create a new folder on your Desktop named FirstWebPage.

2. Open VS Code and select File > Open Folder to open the directory.

3. Create the project files named exactly index.html and style.css.

Step 2: Construct the HTML Skeleton

Open index.html and insert the boilerplate code below. This establishes the link between your content and your stylesheet. 

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Project Started</h1>
</body>
</html>

Step 3: Define Visual Styles

Navigate to style.css and apply these rules to set the background color and typography: 

body {
  background-color: #111;
  color: #ccc;
  font-family: sans-serif;
}
h1 {
  color: #57c8ff;
}

Troubleshooting

If the page is blank, verify you saved the files. If colors aren't changing, ensure the href in your HTML matches your CSS filename exactly.

HTML: Standard language for web content structure.
CSS: Language used for visual layout, colors, and fonts.
