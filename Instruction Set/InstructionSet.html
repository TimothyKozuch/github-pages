The Fundamentals of HTML

HTML (HyperText Markup Language) is the structure/skeleton of the internet. While it is often thought to be a programming language, it is actually a markup language which means it only determines how to display content not handle logic or calculations. It uses a system of tags to annotate elements so that browsers can interperet how to properly display the content of the page.

An HTML element usually consists of an opening tag, the content, and a closing tag (e.g., <tag>content</tag>). However, tags can also have attributes, which provide additional information, such as a source for an image or a destination for a link (e.g. <a href="https://address.com/" class="link">linked text</a>).

Step 1: Create the HTML Foundation

After creating the empty html file it should have automatically opened in VS Code. If it did not open double click on it in the left panel to open it up. In this section we will place the basic boilerplate used in HTML at the top of the file in order for the browser to interpret our page properly.

  1. Place <!DOCTYPE html> at the very top to tell the browser you're using HTML5.
  2. Next use <html lang="en"></html> to wrap our <head>(invisible information) and <body>(visible content) in one element.
  3. To tell the browser what set of characters we are using, the title of the page, and any other metadata that won't actually appear on the page place the <head>.
  4. Place <meta charset="UTF-8"> inside the <head> to ensures special characters and symbols display correctly on your page.
  5. Along with the charset, place <title>My First Page<title> in the <head>. This will set the tab name seen in the browser or search results.
  6. The opposite of <head> is <body>. Place <body> to contain all the content that we want to be displayed on our page (text, images, buttons, etc.).
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>My First Page</title>
    </head>
    <body>

    </body>
</html>

Step 2: Create the Structural Layout

Before adding text, we need to define the containers that will organize your page. We will use a main <div> to center everything, and <section> tags to separate your bio from your contact info.

  1. Inside the <body>, create a <div>.
  2. Inside that div, create three <section> elements.
  3. These sections will eventually hold your Header, Bio, and Links.
<body>
  <div class="container">
    <section id="header">
      </section>

    <section id="bio">
      </section>

    <section id="contact">
      </section>
  </div>
</body>

Step 3: Add Your Personal Information

Now, fill your sections with actual content. This is where you introduce yourself to the world!

  1. Header Section: Add an <h1> for your name and an <img> for your profile picture.
  2. Bio Section: Use an <h2> title and a <p> tag to write a short paragraph about yourself. Use <strong> to highlight key skills.
  3. Contact Section: Use an <a> tag (Anchor) to link to your email or social media, and a <button> for a "Contact Me" call to action.
<section id="header">
    <img src="profile.jpg" alt="My Profile Picture">
    <h1>Jane Doe</h1>
</section>

<section id="bio">
    <h2>About Me</h2>
    <p>I am a <strong>Web Developer</strong> in training!</p>
</section>

<section id="contact">
    <a href="mailto:hello@site.com">Email Me</a>
    <button>Download Resume</button>
</section>

Step 4: Define Visual Styles

Navigate to style.css and apply these rules to set the background color and typography: 

body {
  background-color: #111;
  color: #ccc;
  font-family: sans-serif;
}
h1 {
  color: #57c8ff;
}

Troubleshooting

If the page is blank, verify you saved the files. If colors aren't changing, ensure the href in your HTML matches your CSS filename exactly.

HTML: Standard language for web content structure.
CSS: Language used for visual layout, colors, and fonts.
