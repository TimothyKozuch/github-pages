How to Build a Webpage

An instruction set for creating your first web page with HTML and CSS.

Introduction

This instruction set guides novice users through learning how HTML can be used to create a webpage. It does not include how to host the created webpage, only the writing of the HTML.

Steps Include

  1. Environment Setup: Setting up the development environment
  2. Create the HTML Foundation: Creating the standard HTML boilerplate
  3. Create the Structural Layout: Creating Sections to orginize content
  4. Add Your Content: Place the actual content on the webpage
  5. Style Your Page: Use minimal CSS to style your page
  • Total Time Required: 30–45 minutes
  • Skills Required: Basic file management involving creating, moving, and naming folders.
  • Required Materials:

    1: A modern web browser like Chrome

    2: A code editor like VS Code (you can download VS Code here, and if you need additional help downloading it there are guides on youtube like this)

Step 0: Environment Setup

A: Create a new folder on your Desktop named "FirstWebPage". You can do this by right clicking on your desktop (not an application on the desktop) and selecting new > Folder.You can then type "FirstWebPage" to name it and then press enter to finish creating it.

B: (shown in image) Open VS Code and select File > Open Folder to open the directory (green arrow then red arrow). Navigate to the folder you created on the desktop and open it.

C: (shown in image) Go to extensions on the left hand side of the screen(green arrow), it looks like four boxes, and search and install "Live Server" by Ritwick Dey. After you install it, you can see what your page looks like by clicking go live at the bottom right of VS code(red arrow).

D: Click on File > New File at the top left corner of VS code, like you did to open the folder, to create a new file and name it "index.html".

The Fundamentals of HTML

HTML (HyperText Markup Language) is the structure of the internet. While it is often thought to be a programming language, it is actually a markup language which means it only determines how to display content not handle logic or calculations. It uses a system of tags to annotate elements so that browsers can interperet how to properly display the content of the page.

An HTML element usually consists of an opening tag, the content, and a closing tag (e.g., <tag>content</tag>). However, tags can also have attributes, which provide additional information, such as a source for an image or a destination for a link (e.g. <a href="https://address.com/" class="link">linked text</a>).

Step 1: Create the HTML Foundation

After creating the empty html file it should have automatically opened in VS Code. If it did not open double click on it in the left panel to open it up. In this section we will place the standard boilerplate used in HTML at the top of the file in order for the browser to interpret our page properly.

  1. Place <!DOCTYPE html> at the very top to tell the browser you're using HTML5.
  2. Next use <html lang="en"></html> to wrap our <head>(invisible information) and <body>(visible content) in one element.
  3. To tell the browser what set of characters we are using, the title of the page, and any other metadata that won't actually appear on the page place the <head>.
  4. Place <meta charset="UTF-8"> inside the <head> to ensure special characters and symbols display correctly on your page.
  5. Along with the charset, place <title>My First Page</title> in the <head>. This will set the tab name seen in the browser or search results.
  6. The opposite of <head> is <body>. Place <body> to contain all the content that we want to be displayed on our page (text, images, buttons, etc.).
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>My First Page</title>
    </head>
    <body>

    </body>
</html>

Step 2: Create the Structural Layout

We need to define the containers that will organize and hold the content on your page. We will use a main <div> to center everything, and <section> tags to separate your header, bio, and contact info.

  1. Inside the <body>, create a <div> with the class="container" attribute.
  2. Inside that div, create three <section> elements to hold your Header, Bio, and Links.
  3. label each section with its attribute with id="attribute".
<body>
  <div class="container">
    <section id="header">
    </section>

    <section id="bio">
    </section>

    <section id="contact">
    </section>
  </div>
</body>

Step 3: Add Your Content

Now, place your content in the appropriate sections with the relevant tag. This is where you finally get to see the content on the page!

It is important that you only place information that you are fine with being public on a website that is available to the public.

  1. In your header section place a <h1> to make your name be as large as a header similar to in any word processor.
  2. Next we are going to take any profile picture you like and place it in the "FirstWebPage" folder alongside the "index.html" file.
  3. Also create a <img> tag in the header section and add the src="nameOfProfilePic.jpg" and alt="Profile Picture" attributes to it. You should substitute the name "nameOfProfilePic.jpg" for the name of your picture.
  4. To fill out your bio section use a <h2> tag to make a sub-header that is smaller than your name and a <p> tag to write a short paragraph about yourself. Use multiple <p> tags to create more paragraphs.
  5. In the contact section use an <a> tag to link to your email, social media, or any other site with an address. 
<section id="header">
    <img src="nameOfProfilePic.jpg" alt="Profile Picture">
    <h1>Jane Doe</h1>
</section>

<section id="bio">
    <h2>About Me</h2>
    <p>I am a Web Developer in training!</p>
</section>

<section id="contact">
    <a href="mailto:hello@site.com">Email Me</a>
    <a href="https://www.youtube.com/watch?v=dQw4w9WgXcQ">Video</a>
</section>

Step 4: Style Your Page

CSS changes the look of your page. To use it, place <style> tags inside your <head> section. We will focus on the three most important styles to make your page look professional.

  1. Use the .container selector to set max-width and margin to stop the contents from being too wide and to create a margin between the content and the side of the browser window.
  2. Set the color and background-color properties of h1 and body to create a high-contrast theme that makes your content easy to read against the background. Feel free to change the colors to anything you would like while keeping in mind legibility.
  3. Apply a font-family to the body tag to change the text from the default style of the browser to a cleaner look like sans-serif. This stops unpredictable default behavior from the browser and keeps the page looking the way you like.
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
    
  <style>
    .container {
    max-width: 900px;
    margin: 0 auto;
    }

    h1 {
    color: #57c8ff;
    }

    body {
    background-color: #111;
    color: white;
    font-family: sans-serif;
    }
  </style>
</head>

Troubleshooting

If the page is blank, verify you saved the files.(CTRL+S)

If the page doesn't change try clicking on the port:5500 button on the bottom right of VS code and then relaunching the live server by clicking again.
