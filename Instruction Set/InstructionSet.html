How to Build a Webpage

An instruction set for creating your first web page with HTML and CSS.

Introduction

This instruction set guides novice users through learning how HTML and CSS can be used to create a webpage. HTML(hypertext markup language) and CSS (Cascading Style Sheet) are the foundation of any webpage that you see across the internet.

The Fundamentals of HTML

HTML (HyperText Markup Language) is the structure/skeleton of the internet. While it is often thought to be a programming language, it is actually a markup language which means it only determines how to display content not handle logic or calculations. It uses a system of tags to annotate elements so that browsers can interperet how to properly display the content of the page.

An HTML element usually consists of an opening tag, the content, and a closing tag (e.g., <tag>content</tag>). However, tags can also have attributes, which provide additional information, such as a source for an image or a destination for a link (e.g. <a href="https://address.com/" class="link">linked text</a>).

Step 1: Construct the HTML Skeleton

Open index.html and establish the document's foundation and "behind-the-scenes" information:

  • <!DOCTYPE html>: The declaration that tells the browser you're using HTML5.
  • <html>: The Root Element that wraps every other piece of code.
  • <head>: The Metadata Container for info that doesn't show up on the page.
  • <body>: The Content Area where everything visible (text, images, buttons) lives.
  • <meta charset="UTF-8">: Ensures special characters display correctly.
  • <title>: The Tab Name seen in the browser or search results.
  • <link rel="stylesheet" href="style.css">: The bridge that connects your HTML to your CSS file.

Step 2: Define the Boxes (Containers)

Use containers to group elements and organize the layout of your page:

  • <div>: A generic container used to group elements for styling.
  • <section>: Defines a specific section of content, like an "About" area.
  • <article>: Used for independent, self-contained content like a blog post.
  • <nav>: Tells the browser that this box contains navigation links.

Step 3: Add the Content

Fill your "boxes" with the elements the user will actually see and interact with:

  • Headings (<h1>): Used to show different levels of content hierarchy.
  • <p>: Standard paragraphs used for blocks of text.
  • <strong> & <em>: Formatting tags for **bold** and *italics*.
  • <a> (Anchor): Creates links; requires the href attribute.
  • <img>: Displays pictures using the src attribute.
  • <button>: Creates a clickable button.
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <nav>
        <a href="index.html">Home</a>
    </nav>

    <section>
        <h1>Hello World</h1>
        <p>This is a <strong>bold</strong> start!</p>
        <img src="image.jpg" alt="A nice view">
        <button>Click Me</button>
    </section>
</body>
</html>

Step 3: Define Visual Styles

Navigate to style.css and apply these rules to set the background color and typography: 

body {
  background-color: #111;
  color: #ccc;
  font-family: sans-serif;
}
h1 {
  color: #57c8ff;
}

Troubleshooting

If the page is blank, verify you saved the files. If colors aren't changing, ensure the href in your HTML matches your CSS filename exactly.

HTML: Standard language for web content structure.
CSS: Language used for visual layout, colors, and fonts.
