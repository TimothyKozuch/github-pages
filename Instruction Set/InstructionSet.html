How to Build a Webpage

An instruction set for creating your first web page with HTML and CSS.

Introduction

This instruction set guides novice users through learning how HTML and CSS can be used to create a webpage. HTML(hypertext markup language) and CSS (Cascading Style Sheet) are the foundation of any webpage that you see across the internet.

  • Total Time: __ minutes
  • Skills Required: Basic computer literacy and ability to manage files in the file exploror.
  • Required Materials:

    1: A modern web browser like chrome

    2: A code editor like VS Code

    (you can download it here and if you need additional help there are guides on youtube like this)

Step 1: Environment Setup

1. Create a new folder on your Desktop named FirstWebPage. You can do this by right clicking on your desktop (not an application on the desktop) and selecting new > Folder.You can then type "FirstWebPage" to name it and then press enter to finish creating it.

2. Open VS Code and select File > Open Folder to open the directory. Navigate to the folder you created on the desktop and open it.

3. Click on File > New File at the top left corner of VS code to create two files and name them "index.html" and "style.css".

Step 2: Construct the HTML Skeleton

Open index.html and insert the boilerplate code below.

This establishes the link between your content and your stylesheet.

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Project Started</h1>
</body>
</html>

Step 3: Define Visual Styles

Navigate to style.css and apply these rules to set the background color and typography: 

body {
  background-color: #111;
  color: #ccc;
  font-family: sans-serif;
}
h1 {
  color: #57c8ff;
}

Troubleshooting

If the page is blank, verify you saved the files. If colors aren't changing, ensure the href in your HTML matches your CSS filename exactly.

HTML: Standard language for web content structure.
CSS: Language used for visual layout, colors, and fonts.
