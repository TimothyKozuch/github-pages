Loop-Hole

Language: Python (Pygame)  |  GMTK 2025 Game Jam, Summer 2025

View on GitHub Download Source (.zip)

Description

A 2D platformer built for the GMTK 2025 Game Jam, created as team leader. Features robust collision detection, physics, branching dialogue trees driven by external JSON files, and handmade sprite animations.

Key Features

  • Time-loop game mechanic as core gameplay
  • Robust collision detection and physics system
  • Branching dialogue trees loaded from JSON files
  • Handmade animations for player and enemy characters
  • Team leadership: task delegation and teaching Pygame fundamentals
  • Clean GitHub version control throughout development
