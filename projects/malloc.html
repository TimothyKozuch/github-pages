Malloc

Language: C

View on GitHub Download Source (.zip)

Description

A custom dynamic memory allocator implementing malloc, free, and realloc in C. Manages a heap with explicit free lists using first-fit placement, block splitting, and coalescing of adjacent free blocks to reduce fragmentation.

Key Features

  • First-fit free block placement
  • Block splitting to minimize wasted space
  • Immediate coalescing of adjacent free blocks
  • Explicit free list with header/footer boundary tags
