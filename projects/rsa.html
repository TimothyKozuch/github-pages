RSA Encryption

Language: Python

Description

An implementation of the RSA public-key cryptography algorithm in Python. Generates key pairs, encrypts plaintext messages, and decrypts ciphertext using modular exponentiation and the mathematics of prime factorization.

Key Features

  • Prime number generation and key pair creation
  • Modular exponentiation for fast encryption/decryption
  • End-to-end encrypt → decrypt demonstration
