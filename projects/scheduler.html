Course Scheduler

Language: Java  |  Course: Object Oriented Programming with Web Applications, PSU Spring 2024

View on GitHub Download Source (.zip)

Description

A course scheduling application with separate administrator and student roles. Administrators can create and manage course offerings; students can browse and enroll. Built with a Java Swing front-end GUI and a SQL database back end.

Key Features

  • Role-based access: administrator and student views
  • Java Swing GUI for desktop interaction
  • SQL database for persistent course and enrollment data
  • Enrollment conflict detection
